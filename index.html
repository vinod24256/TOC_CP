🌳 Non-Deterministic TM (NTM)Variant
NTM explores all possible transitions simultaneously. For decryption with unknown key, NTM tries all 26 possible shifts in parallel — if ANY branch finds the correct decryption, it accepts. Theorem: Every NTM has an equivalent deterministic TM (exponential blowup possible).
"If one branch accepts, the NTM accepts — this models brute-force cipher key search."
✠ Church-Turing Thesis
Claim: Every effectively computable function is Turing-computable.

The Caesar cipher TM above is not just a simulation — it represents the most general model of computation. WhatsApp's real encryption stack (AES-256-CBC, HMAC-SHA256, Curve25519 ECDH via Signal Protocol) — however complex — is still just a Turing Machine operating on bits.

There is no more powerful model of computation than the Turing Machine. All modern computers, quantum computers, and neural networks are Turing-equivalent (under computability — not complexity).
