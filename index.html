How it Works

01

Message enters the tape

Your plaintext is written cell by cell. Head positions at 0. State: q₀.

┌───┬───┬───┬───┬───┐
│ □ │ h │ e │ l │ l │
└───┴───┴───┴───┴───┘
02

Transitions fire — characters encrypt

δ(q, σ) → (q', σ', D) fires for each character. Uppercase → q_upper. Lowercase → q_lower. Each shifts by k=3.

[q₀] ──'h'/'k',R──▶ [q_lower]
03

TM halts — message delivered

On blank □, transitions to q_accept. Tape holds ciphertext. L(M) = Σ* is Recursive. Time: O(n).
